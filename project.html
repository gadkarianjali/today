PROJECTS

Title: Online Food Ordering Website Duration: 1 month Objective: To create an interactive and responsive platform for users to browse menus, place orders, and make payments online. Technologies Used: HTML, CSS, JavaScript, Node.js, MongoDB. Details: Developed a user-friendly front-end interface using React. Implemented back-end APIs in Node.js for user authentication, order processing, and payment gateway integration. Designed the database schema using MongoDB and integrated real-time updates for order tracking. Outcome: Successfully created a platform used by a small business for order management, increasing efficiency by 40%. Role: Full-stack developer handling both front-end and back-end components.
